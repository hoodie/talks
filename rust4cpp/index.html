Main Goals

  1. Performant
  2. Safe
  3. Productive / Ergonomic

Performant

  • zero cost abstractions
  • close to the hardware
  • no GC
  • no runtime
  • system threads
  • async/await
  • C FFI (bindgen both directions)

Safe

  • no unintialized variables
  • no UB
  • strong types that guarantee
    • no race conditions
    • clear ownership/lifetimes
    • when to copy/move
  • immutability
  • mutexes are containers

Ergonomic

  • cargo: build test doc bench publish
  • modules
  • pattern matching
  • optionals
  • friendly compiler
  • great 3rd party crates

traits!

pub struct NewsArticle {
    pub headline: String,
    pub date: String,
    pub author: String,
}

fn printSummary(s: &dyn Summary) {
    println!("{}", s.summarize());
}

pub trait Summary {
    fn summarize(&self) -> String;
}

impl Summary for NewsArticle {
    fn summarize(&self) -> String {
        format!("{}, by {} ({})", self.headline, self.author, self.date)
    }
}

}
